# Formatics Examples

This directory contains example scripts demonstrating Formatics concepts and usage.

## Examples

### `example_leap_aware.py`
Demonstrates filestate folderleap functionality - tracking when a script moves between directories.

### `example_slot_orbit.py`
Demonstrates basic Slot and Orbit usage with error handling.

### `example_form_composition.py`
Demonstrates Element and FormElement composition with transformations.

## Running Examples

From the repository root:

```bash
# Run a specific example
python examples/example_slot_orbit.py

# Or using the module
python -m examples.example_slot_orbit
```
